	四、编程规范
	 STM32与PC通讯协议
	1. 语音播报：
	STM命令 ：
	"StartPam%"+txt+"%$$"

	2. 读卡：
	STM命令：
	"ReadCard%$$"
	PC返回：
	成功："ReadSucess%"+txt+"%$$"		正确传数据
	失败："ReadCardFail%"+txt+"%$$"	错误传播报文本

	3. 写卡：
	STM写4442命令：
	"Write4442Card%"+txt+"%$$"
	PC返回：
	4442成功："Write4442Sucess%$$"
	失败："Write4442Fail%$$"

	STM写4428命令：
	"Write4428Card%"+txt+"%$$"
	PC返回：
	4428成功："Write4428Sucess%"+txt+"%$$"
	4428失败:   "Write4428Fail%$$"

	4. 握手机制：
	STM323发：Hello%$$
	PC回：HelloToo%$$（握手时间建议3分钟）

	5. 关机命令 ：
	STM32发：
	"ShutDown%$$"
